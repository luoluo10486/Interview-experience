# 中科江南数字信息技术 - 技术岗面经

> 面试时间：2026-05-28
>
> 说明：根据当天面试记录截图整理，尽量保留原始问答内容，方便后续复盘。

## 一、面试整体感受

- 风格偏日常聊天，不是特别压迫。
- 问题以项目为主，基本是顺着项目经历一路追问。
- 实习相关问得比较轻，没有深挖。
- 八股也有，但整体不算特别刁钻。

## 二、面试问题记录

### 1. 项目经历

- 项目条条问，主要是围绕整个流程展开。
- 重点会追问你在项目里具体负责了什么、为什么这么设计、流程怎么跑通。

### 2. 实习经历

- 只简单问了一下实习做了什么。
- 没有细问太深，更像是确认你是否真正参与过业务。

### 3. Java 基础

问到的基础题包括：

- Java 的数据结构
- Bean 的生命周期
- 循环依赖

### 4. 数据库

- 问了除了 MySQL 之外还用过什么数据库。
- 这类问题主要是在看你对持久化方案和数据库选型有没有实际接触。

## 三、反问环节

反问了这些内容：

- 业务方向
- 上班时间

对方给出的信息大致是：

- 早九晚六
- 午休两个半小时
- 节假日正常放
- 除元旦外，其他节假日会调休

## 四、薪资信息

- 薪资要看 HR 进一步沟通。
- 转正后薪资据说能过 `w`。

## 五、复盘建议

- 项目部分要准备成“业务场景 - 技术方案 - 为什么这么设计 - 最后效果”四步讲法。
- Java 基础要把高频八股再顺一遍，尤其是：
  - `HashMap` / `ArrayList` 等常用数据结构
  - Bean 生命周期
  - 循环依赖和三级缓存
- 数据库问题除了 MySQL，也要准备一两个你真实用过的数据库，并能说清楚使用场景。
